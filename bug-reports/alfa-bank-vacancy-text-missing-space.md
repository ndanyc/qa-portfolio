# Баг-репорт: Массовое отсутствие пробелов в разделе «Что мы предлагаем».

**Ссылка на страницу:** [https://job.alfabank.ru/vacancies/sankt-peterburg/remote-job/qa-fullstack-(manual+at)_35621?city=0169]

**Серьезность (Severity):** Minor (Незначительный)

**Окружение:**
- ОС: Windows 11 Pro, ver. 25H2 ( сборка: 26200.8655 )
- Браузер: Yandex Browser ver. 26.6.1.1039 (64-bit)
- Устройство: Desktop

**Шаги к воспроизведению:**

1. Перейти на страницу вакансии QA Fullstack (Manual & AT) по ссылке (прикреплена выше).
2. Спуститься до раздела «Что мы предлагаем».
3. Прочитать любую из строк списка.

**Фактический результат:**

Во всех строчках раздела «Что мы предлагаем» отсутствуют пробелы между словами. Текст отображается слитно, что затрудняет его чтение.
Ниже приведены конкретные примеры ошибок:

- `Стабильныйи прозрачный доход: размер заработной платы обсуждается по итогамсобеседования` (слитно: «Стабильныйи ... по итогамсобеседования»)
- `Гибкийграфик работы: вы сможете планировать время так, как удобно вам и вашейкоманде` (слитно: «Гибкийграфик .... вашейкоманде»)
- `Полнуюудаленку или гибрид на выбор` (слитно: «Полнуюудаленку»)
- `Сложныеи интересные задачи` (слитно: «Сложныеи»)
- `Заботуо вашем здоровье: ... куда входитстоматология, ... страхование икомпенсация` (слитно: «Заботуо», «входитстоматология», «икомпенсация»)
- `Возможностьвертикального и горизонтального карьерного роста: ... регулярно проходяттренинги` (слитно: «Возможностьвертикального», «проходяттренинги»)
- `Оплатупосещения профильных конференций ... кпубличным выступлениям` (слитно: «Оплатупосещения», «кпубличным»)
- `Доступк бесплатным корпоративным библиотекам ... ибизнес-изданий` (слитно: «Доступк», «ибизнес-изданий»)

**Ожидаемый результат:**

Все слова в списке должны быть разделены корректными пробелами. Текст должен быть оформлен грамотно и читабельно. 
Пример правильного отображения: «Стабильный прозрачный доход: размер заработной платы обсуждается по итогам собеседования».
