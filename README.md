---

### 📂 Структура портфолио

*   **[`/test-cases`](./test-cases)** — Учебные чек-листы и тест-кейсы.
*   **[`/bug-reports`](./bug-reports)** — Примеры оформления баг-репортов.
*   **[`/web3-qa-portfolio`](./web3-qa-portfolio)** — **Ключевой раздел.** Мой опыт тестирования криптопротоколов (тестнеты, UX, развертывание нод, взаимодействие с командами проектов), а также план по поиску багов в Web3-стартапах.
